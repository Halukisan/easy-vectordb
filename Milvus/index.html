Milvus 教程 ​

欢迎来到 Milvus 向量数据库教程！本教程将带你从零开始，系统学习 Milvus 的核心概念、部署方法和实际应用。

学习内容 ​

📖 基础理论 ​

  • 前言 - 了解向量数据库的意义和应用场景
  • Milvus 介绍 - 深入了解Milvus的架构和特性
  • milvus 索引介绍 - 向量索引的原理和选择
  • 聚类算法介绍 - 聚类算法在向量检索中的应用

🚀 部署实践 ​

  • Milvus Lite 部署与应用 - 轻量级本地部署

  • milvus 文本嵌入实战 - 文本向量化实践
  • milvus pdf 嵌入实战 - PDF文档处理
  • milvus pdf 多模型嵌入实战 - 多模态向量化
  • milvus 数据切分总结 - 数据预处理策略

🎯 优化与实践 ​

  • milvus 存储优化 - 存储性能优化
  • GPU 加速检索 - 基于 FusionANNS - GPU加速方案
  • Limit - 向量限制
  • Meta-Chunking：一种新的文本切分策略 - 先进的文本处理

快速开始 ​

如果你是第一次接触向量数据库，建议按以下顺序学习：

  1. 理解概念 - 从前言开始，了解向量数据库的基本概念
  2. 动手实践 - 通过Milvus Lite 部署与应用快速体验
  3. 深入学习 - 学习完整的Standalone 部署
  4. 实战应用 - 查看实战项目了解真实应用场景
